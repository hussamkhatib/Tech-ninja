# TECH NINJA
A quiz consisting of JavaScript code snippets.

### Technologies used 
* React 
* React hooks
* styled-components
* Netlify

### Steps to run it locally run locally 
1 Clone the repository
```
git clone https://github.com/hussamkhatib/Tech-ninja.git
cd Tech-ninja
```
2 install dependencies  
`npm install`  
3 start local server  
`npm start`
